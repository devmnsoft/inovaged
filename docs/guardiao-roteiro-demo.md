# Roteiro de demonstração — InovaGED Guardião

1. Abrir um documento real no GED.
2. Acessar `/DocumentGuardian/{documentId}`.
3. Mostrar cabeçalho, pasta, tipo, completude e risco.
4. Mostrar alerta com regra/versionamento e evidências.
5. Mostrar documentos relacionados e obrigações.
6. Mostrar linha do tempo probatória.
7. Mostrar histórico de decisões humanas.
8. Abrir logs/auditoria e localizar `DOCUMENT_GUARDIAN_VIEW` pelo documento/correlationId.
