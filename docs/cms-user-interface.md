# CMS User Interface

A interface `Signature/Cryptographic` deve separar registro interno operacional de assinatura CMS. O registro interno não é assinatura criptográfica, não exige CPF manual e usa usuário autenticado. O fluxo CMS deve mostrar agente, pairing, certificados, sessão, confirmação, assinatura, validação, status segregados e downloads `.p7s`/pacote.
