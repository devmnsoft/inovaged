# Template de Plano de Recuperação de Desastres

- Nome, tenant/escopo, versão, responsável e substituto.
- Sistemas críticos, dependências, RPO, RTO e ordem de recuperação.
- Contatos, pré-requisitos, procedimentos, critérios de ativação e retorno.
- Último teste, resultado, pendências, próxima revisão e status.

Este template documenta o plano; não representa redundância, HA ou failover automático.
